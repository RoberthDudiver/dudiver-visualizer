## Español

### Que es?

**Dudiver Visualizer Studio** es una herramienta gratuita para crear lyric videos profesionales con sincronizacion automatica usando inteligencia artificial (Whisper).

### Funciones

- **Sincronizacion automatica** con Whisper AI (palabra por palabra)
- **Multiples formatos**: YouTube (1920x1080), TikTok (1080x1920), Instagram (1080x1080), Stories
- **Efectos visuales**: particulas, ondas, viñeta, glow en texto, barra de progreso
- **Fondo personalizado**: imagen o video
- **Modo transparente (alpha)**: exporta WebM con fondo transparente para editar en cualquier programa
- **Spot publicitario**: agrega una pantalla final con texto, imagen o video (fade-in profesional)
- **Esquemas de colores**: Noche, Fuego, Oceano, Neon, Oro
- **Preview en tiempo real**: ve como queda antes de generar
- **Seleccion de duracion**: 30 seg, 1 min, 2 min, 3 min o completo

### Instalacion

1. Descarga el instalador desde [Releases](../../releases)
2. Ejecuta `DudiverVisualizerSetup.exe`
3. Listo!

## English

### What is it?

**Dudiver Visualizer Studio** is a free tool for creating professional lyric videos with automatic AI synchronization (Whisper).

### Features

- **Automatic sync** with Whisper AI (word by word)
- **Multiple formats**: YouTube (1920x1080), TikTok (1080x1920), Instagram (1080x1080), Stories
- **Visual effects**: particles, waves, vignette, text glow, progress bar
- **Custom background**: image or video
- **Transparent mode (alpha)**: export WebM with transparent background for editing
- **Ad spot**: add a final screen with text, image, or video (professional fade-in)
- **Color schemes**: Night, Fire, Ocean, Neon, Gold
- **Real-time preview**: see how it looks before generating
- **Duration selection**: 30 sec, 1 min, 2 min, 3 min, or full

### Installation

1. Download the installer from [Releases](../../releases)
2. Run `DudiverVisualizerSetup.exe`
3. Done!
